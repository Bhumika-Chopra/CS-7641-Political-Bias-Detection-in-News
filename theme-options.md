# Colorful Jekyll Theme Options

Here are colorful themes you can use by just changing one line in `_config.yml`:

## Current Theme: Cayman
```yaml
remote_theme: pages-themes/cayman@v0.2.0
```
- Blue gradient header
- Clean, professional look
- Good for academic projects

## Alternative Colorful Themes:

### Leap Day (Green/Colorful)
```yaml
remote_theme: pages-themes/leap-day@v0.2.0
```
- Green color scheme
- Vibrant and modern

### Tactile (Orange/Colorful)
```yaml
remote_theme: pages-themes/tactile@v0.2.0
```
- Orange and colorful design
- Great for portfolios

### Midnight (Dark with Colors)
```yaml
remote_theme: pages-themes/midnight@v0.2.0
```
- Dark theme with colorful accents
- Modern and sleek

### Time Machine (Retro Colors)
```yaml
remote_theme: pages-themes/time-machine@v0.2.0
```
- Retro color scheme
- Unique vintage look

## How to Change:
1. Edit `_config.yml`
2. Change the `remote_theme:` line
3. Push to GitHub
4. Wait 2-5 minutes for the site to rebuild

All these themes work with GitHub Pages and require no custom CSS!
